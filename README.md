## Tech Stack

### Backend
- **Node.js** + **Express** - Server
- **MongoDB** + **Mongoose** - Database
- **JWT** & **Passport.js (Google OAuth2)** - Authentication & Sessions
- **Bcryptjs** - Password hashing
- **CORS** - Cross-origin requests
- **Joi** - Input validation
- **Google Generative AI (Gemini)** - AI Career Advisor Engine

### Frontend
- **React** 19 - UI Framework
- **React Router** - Navigation
- **Axios** - HTTP Client
- **Bootstrap 5** - Styling
- **Recharts** - Data visualization

## Latest Features
- **AI Career Advisor**: Integration with the Google Gemini API to provide personalized career recommendations based on user skills.
- **Google OAuth Login**: Secure authentication flow using Google Identity Services.
- **Skill Level Tracking & Analytics**: Comprehensive dashboard showing compatibility matches and areas for improvement.
